Instalación

Windows

Descomprime la carpeta en Archivos de Programa. Para abrir la aplicación entra en la carpeta y abre WifiUC.exe

Mac OS X

Descomprime el archivo zip y arrastra la aplicaciones Wifi UC a tu carpeta de Aplicaciones

Linux

Descomprime la carpeta donde quieras. Para abrir la aplicación entra en la carpeta y ejecuta el archivo Wifi UC
